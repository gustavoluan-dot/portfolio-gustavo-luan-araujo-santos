# ⚙️ Engenharia de Software com Bubble.io

## 📝 Descrição do Projeto
Este diretório contém os resultados e a documentação do projeto focado na aplicação de princípios de Engenharia de Software utilizando a plataforma de desenvolvimento no-code **Bubble.io**. O objetivo central consistiu na modelação e criação de um *Dashboard* (painel de controlo) interativo e otimizado, destinado à gestão ágil de informações empresariais, como o acompanhamento de propostas comerciais e orçamentos.

O trabalho demonstra como as metodologias clássicas de engenharia de software, incluindo o levantamento de requisitos, a estruturação de bases de dados e a arquitetura de sistemas, podem ser aplicadas de forma rigorosa em ambientes de programação visual.

## 📂 Conteúdo do Diretório
* `Entrega Final (1).pdf`: Documentação técnica detalhada, contendo os requisitos do sistema, o mapeamento dos fluxos de trabalho (*workflows*) e a análise arquitetural da solução implementada.

## 🚀 Tecnologias e Metodologias
* **Plataforma:** Bubble.io (Desenvolvimento Visual / No-Code).
* **Conceitos Aplicados:** Engenharia de Software, Modelação de Dados Relacionais, Criação de Interfaces de Utilizador (UI) e Lógica de Negócio.

## 📊 Resultados e Aprendizagens
A abstração do código tradicional permitiu um foco muito mais denso na lógica de negócio e na arquitetura da solução. Os principais marcos do projeto incluem:
* **Desenvolvimento Ágil e Eficiente:** Rápida prototipagem e implementação de um painel de controlo altamente funcional e responsivo.
* **Estruturação de Lógica Complexa:** Criação de *workflows* otimizados para manipulação de dados dinâmicos em tempo real e atualização de estados visuais.
* **Foco em Escalabilidade:** Aplicação de boas práticas de engenharia para garantir que a aplicação construída em Bubble.io mantenha a performance e a organização estrutural à medida que novas funcionalidades sejam necessárias.
