# 🛒 ZaestraVitta: E-commerce de Produtos de Limpeza

## 📝 Descrição do Projeto
O **ZaestraVitta** é um projeto de desenvolvimento web focado na criação de uma interface de e-commerce para a comercialização de produtos de limpeza. Estruturado como um projeto acadêmico de base na arquitetura front-end, o site simula uma vitrine digital completa, priorizando usabilidade, estruturação semântica e organização visual.

Este projeto representa o domínio prático dos fundamentos de marcação de hipertexto e estilização em cascata, requisitos essenciais para a construção de interfaces web consistentes, responsivas e focadas na experiência do usuário.

## 🚀 Tecnologias Utilizadas
* **Linguagem de Marcação:** HTML5 (Foco em estruturação semântica e acessibilidade).
* **Estilização:** CSS3 (Posicionamento, box model, tipografia e aplicação da identidade visual da marca).

## 📊 Objetivos e Aprendizados
A construção da interface do ZaestraVitta permitiu a consolidação de conceitos centrais da engenharia front-end, operando "vanilla" (sem frameworks externos):
* **Estruturação Semântica:** Aplicação rigorosa de tags HTML5 para garantir uma hierarquia de informação lógica e manutenível.
* **Design de Interface (UI):** Criação de um layout intuitivo e limpo para facilitar a navegação do usuário no catálogo de produtos.
* **Fundamentos de CSS:** Domínio sobre regras de especificidade, seletores e organização de estilos para garantir a integridade visual da aplicação.
